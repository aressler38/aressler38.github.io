    Projects

    • Factor Trinomials

      This is a web app that I ported to Android using Apache Cordova. It randomly generates a quadratic of the form ax2+bx+c and asks you to factor it. It is supported by touch/click area hints and a short tutorial.

    • HardWorker

      This is an library for WebWorkers. It allows you to load 'tasks' into a worker context, execute them on demand, and pass/recieve data.

    • Fractals

      This project uses the HardWorker project to run a task in a WebWorker and compute lots of pixel data.

    • Oscillators

      This uses the WebAudio API and allows you to hook up several oscillator nodes to gain nodes and hear the result.

    • Excerpts

      This is a simple demo of a Flask app configured for Apache's mod_wsgi plugin. It utilizes sqlalchemy connected to MySQL for ORM support. It allows users to login, and save text created on a wysiwyg editor to the database.
